# Ecommerce Listing Optimizer Lite — Free Claude Skill

Generate optimized Amazon product listings from raw product info. SEO-optimized titles, bullet points, descriptions, and backend keywords.

## What This Does (Lite — Amazon Only)

Provide your product details and get:
- **Optimized Title** — keyword-rich, character-limit compliant
- **5 Bullet Points** — benefit-driven, scannable, keyword-integrated
- **Product Description** — persuasive HTML-formatted copy
- **Backend Search Terms** — 250-byte optimized keyword list
- **3 Title Variants** for A/B testing

## Quick Start

1. Download the `.skill` file from this repo
2. Install in Claude
3. Describe your product (name, features, target audience, key benefits)
4. Get a complete, optimized Amazon listing ready to publish
